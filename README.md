# 🚀 Guddu Sahu Portfolio

Welcome to my personal portfolio repository! This is a high-performance, premium-design digital space built to showcase my creative experiments and development skills.

## ✨ Live Demo
Check out the live site here: [**saahu.in**](https://www.saahu.in/)

## 🛠️ Tech Stack
- **Frontend**: React.js & Vite
- **Animations**: Framer Motion
- **Styling**: Tailwind CSS & Vanilla CSS
- **Backend/DB**: Firebase Firestore
- **Deployment**: Vercel

## 🌟 Key Features
- **Dynamic Projects**: Projects are managed in real-time via a custom Firebase backend.
- **Vibe-Heavy UI**: Focused on smooth transitions and a premium aesthetic.
- **AI Integration**: Features a custom AI-powered project describer.
- **Fully Responsive**: Optimized for every device from mobile to ultra-wide displays.

## 📁 Repository Structure
- `/src`: Contains all React components and logic.
- `firestore.rules`: Security rules for the database.
- `vercel.json`: Deployment configuration for Vercel.
